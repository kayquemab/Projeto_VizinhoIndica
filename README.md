# 📌 Vizinho Indica

Marketplace comunitário desenvolvido para conectar moradores a serviços confiáveis dentro de condomínios e comunidades, com foco em indicações, avaliações e comunicação entre vizinhos.

## 🚀 Tecnologias

* React
* TypeScript
* Vite
* TanStack Start
* Tailwind CSS
* Supabase
* PostgreSQL

## ✨ Funcionalidades

* Página inicial com listagem de serviços
* Busca por serviços e categorias
* Cadastro e autenticação de usuários
* Criação, edição e gerenciamento de anúncios
* Upload de imagem para anúncios
* Sistema de favoritos
* Página de detalhes do serviço
* Avaliações de serviços
* Mensagens entre usuários
* Dashboard do usuário
* Layout responsivo
* Componentes reutilizáveis
* Integração preparada com Supabase
* Página de erro 404

## ⚙️ Como executar

```bash
git clone https://github.com/kayquemab/Projeto_VizinhoIndica
cd Projeto_VizinhoIndica

# Instalar dependências
npm install

# Executar o projeto
npm run dev
```

## 🔐 Variáveis de ambiente

Crie um arquivo `.env` na raiz do projeto com:

```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

## 📌 Observações

* Projeto desenvolvido com React + Vite + TS
* Utiliza Tailwind CSS para estilização
* Integração com Supabase para autenticação, banco de dados e storage
* Banco de dados planejado com PostgreSQL via Supabase
* Ideal para fins de estudo, prática e portfólio
